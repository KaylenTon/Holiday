# Holiday R Package
I created a custom holiday-themed package.

### The Holiday package provides:
- Pre-made holiday color palettes (discrete and continuous) for ggplot2.
- Holiday-themed ggplot2 functions
- A pseudo dataset (toymaker) for experimenting with visualizations and regression modeling.
- Countdown functions to visualize the number of days until holidays.
- Etc.

### To install from GitHub:
```{r}
# Install and load devtools if needed
install.packages("devtools")
library(devtools)

# Install your package from GitHub
devtools::install_github("Kaylenton/Holiday")
```

### To load the package in RStudio:
```{r}
library(Holiday)
```
